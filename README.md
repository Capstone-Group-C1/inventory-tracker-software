# inventory-tracker-software
Software for the Raspberry Pi to control the inventory tracker system.


## Setup
To setup the virtual environment to run on your machine, follow these steps:

### 1. Install [Miniconda](https://www.anaconda.com/docs/getting-started/miniconda/install)


### 2. Set up the environment using this command:
```bash
conda env create -f environment.yml
```

### 3. Activate the conda environment using this command:
```bash
conda activate aim
```

### 4. Once done making changes to the codebase, deactivate the conda environment using this command:
```bash
conda deactivate
```
